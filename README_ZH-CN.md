## 特性

- 实时预览

- 支持 Gtihub markdown 特性

- 支持 todo lists

- 支持 unicode 表情:kissing_heart: :dart:

- 优美的布局

- 离线编辑

- 支持保存为 HTML 文件

- 支持保存为 markdown 文件

- 支持白天/夜间两种主题

- 支持全屏阅览/编辑

## Run and Build

```shell
git clone https://github.com/huruji/SilkDown.git

cd SilkDown & yarn

yarn run dev

// or

yarn run build
```
